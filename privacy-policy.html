Resume Medis Helper RSBHHB

Terakhir diperbarui: 15 Juli 2026

Ringkasan

Resume Medis Helper RSBHHB adalah ekstensi Chrome untuk membantu user membuat dan mereview resume medis pada sistem eRM internal. Ekstensi ini bekerja atas perintah user, misalnya saat user menekan tombol untuk menarik data SOAP, CPPT, penunjang, menjalankan AI, atau memasukkan hasil ke resume.

Data yang Dapat Diproses

Ekstensi dapat membaca dan memproses data yang tampil pada halaman eRM yang sedang dibuka oleh user, termasuk:

  • Data SOAP, CPPT, resume medis, diagnosis, terapi, tindakan, konsultasi, tanda vital, dan pemeriksaan penunjang.
  • Isi PDF hasil penunjang yang dibuka melalui alur penunjang.
  • Draft lokal yang dibuat dari pekerjaan user di side panel.
  • Pengaturan user seperti pilihan provider AI, model, API key pribadi, dan sesi admin.

Tujuan Penggunaan Data

Data digunakan hanya untuk fungsi ekstensi, yaitu:

  • Menarik dan menampilkan data medis dari halaman eRM aktif.
  • Menyusun ringkasan CPPT, SOAP, penunjang, analisa klaim, dan bantuan diagnosis berbasis AI bila user menjalankan fitur tersebut.
  • Mengisi field resume medis atau RM 07 setelah user menekan tombol yang sesuai.
  • Menyimpan draft lokal agar pekerjaan user tidak hilang saat berpindah halaman pasien.
  • Mengelola akses admin, konfigurasi provider AI admin, dan knowledge BPJS melalui backend Supabase.

Penyimpanan Data

  • Draft pasien disimpan secara lokal di browser user menggunakan IndexedDB.
  • Pengaturan user dan API key pribadi disimpan secara lokal di browser user menggunakan chrome.storage.local.
  • API key pribadi tidak dikirim ke server pengelola ekstensi, kecuali digunakan langsung oleh browser user untuk memanggil provider AI yang dipilih user.
  • Konfigurasi API key admin, user admin, dan knowledge BPJS disimpan pada Supabase untuk kebutuhan fitur admin.

Pengiriman Data ke Pihak Ketiga

Ekstensi dapat mengirim data klinis yang relevan ke layanan AI hanya saat user menjalankan fitur AI. Layanan AI yang dapat digunakan bergantung pada pengaturan user atau admin, misalnya Gemini, provider OpenAI-compatible, Sumopod, AImurah, X5Lab, custom endpoint, atau jalur admin melalui Supabase Edge Function.

Data dikirim untuk menghasilkan output yang diminta user, seperti ringkasan, review, analisa, atau saran diagnosis. Ekstensi tidak menjual data user dan tidak menggunakan data untuk iklan.

PDF Penunjang

Untuk fitur penunjang, ekstensi membaca PDF menggunakan parser lokal di browser bila memungkinkan. Hasil ekstraksi yang relevan dapat digunakan sebagai konteks AI setelah user menjalankan fitur rangkuman penunjang.

Izin Chrome yang Digunakan

  • sidePanel: menampilkan panel bantuan resume medis.
  • storage: menyimpan pengaturan dan draft lokal.
  • activeTab dan scripting: membaca atau mengisi data pada halaman eRM aktif setelah user menjalankan fitur.
  • host permissions: diperlukan agar ekstensi dapat bekerja pada alamat eRM internal yang digunakan rumah sakit.

Retensi dan Penghapusan

Draft lokal berada di perangkat/browser user dan dapat diganti, dipulihkan, atau dihapus melalui alur ekstensi. User juga dapat menghapus data lokal melalui pengaturan browser Chrome. Data admin dan knowledge BPJS dikelola oleh admin aplikasi.

Keamanan

Ekstensi membatasi pemrosesan data untuk kebutuhan fitur yang dijalankan user. Namun, karena fitur AI dapat melibatkan provider pihak ketiga, user dan admin harus memastikan penggunaan provider AI sesuai kebijakan internal rumah sakit dan peraturan privasi data yang berlaku.

Perubahan Kebijakan

Kebijakan ini dapat diperbarui bila terdapat perubahan fitur, provider, atau alur penyimpanan data. Versi terbaru akan dipublikasikan pada halaman ini.

Kontak

Untuk pertanyaan terkait privasi atau penggunaan ekstensi, hubungi pengelola internal Resume Medis Helper RSBHHB.
